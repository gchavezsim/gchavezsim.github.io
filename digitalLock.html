Digital FPGA Lock

Welcome to the Digital Lock project! This engaging endeavor allowed me to showcase my skills in VHDL programming and digital circuit design using the Vivado Software and Basys 3 board. With a given state diagram representation and digital circuit design, I successfully implemented a fully functional digital lock on the Basys 3 board.

The digital lock operated based on a predefined state diagram that specified how the lock should function. Using VHDL programming, I meticulously translated the state diagram into a working digital circuit. The Vivado Software provided a powerful platform for designing and simulating the digital lock, ensuring a seamless integration of hardware and software components.

On the Basys 3 board, users could interact with the digital lock by pressing the correct button combination, following the state transitions defined in the state diagram. When the correct sequence was entered, the leftmost LED on the Basys 3 board indicated that the lock was successfully unlocked. Conversely, if an incorrect combination was entered, the rightmost LED indicated that the lock remained locked.

The project's successful implementation demonstrated my ability to apply theoretical concepts of digital circuit design and VHDL programming to create a practical and functional system. The Digital Lock project served as a valuable opportunity to enhance my skills in FPGA development, logic design, and hardware-software integration.

Key Components and Features:

  • VHDL Programming: I utilized VHDL programming to implement the digital lock based on the given state diagram.
  • Vivado Software: The Vivado Software provided a robust environment for designing, simulating, and deploying the digital lock onto the Basys 3 board.
  • Basys 3 Board: The Basys 3 board served as the hardware platform for the digital lock, allowing user interaction through button inputs and LED outputs.
  • User Interaction: Users could input the correct button combination to unlock the digital lock.
  • LED Indicators: The leftmost LED indicated the unlocked state, while the rightmost LED indicated the locked state of the digital lock.
